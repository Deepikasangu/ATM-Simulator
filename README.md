# ATM Simulator (Python)

A console-based ATM system implemented using Object-Oriented Programming (OOP) principles.  
This project simulates basic ATM operations like balance inquiry, deposit, withdrawal, and PIN validation.  
Designed with clean and modular code, this project helps beginners practice Python, OOP, and file handling.
